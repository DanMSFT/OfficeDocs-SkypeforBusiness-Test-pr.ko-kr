# 영구 채팅 서버 Cmdlet

 

_**마지막으로 수정된 항목:** 2012-06-27_

영구 채팅 cmdlet을 사용하면 Microsoft Lync Server 2013 영구 채팅 서비스(이전의 그룹 채팅 서비스)를 관리하고 구성할 수 있습니다. 영구 채팅을 통해 사용자는 온라인 인스턴트 메시징 세션에 참가할 수 있습니다. 이러한 세션은 실시간으로 수행되지만 각 세션의 콘텐츠가 영구 저장됩니다. 즉, 대화를 누구나 언제라도 다시 시작할 수 있습니다.
