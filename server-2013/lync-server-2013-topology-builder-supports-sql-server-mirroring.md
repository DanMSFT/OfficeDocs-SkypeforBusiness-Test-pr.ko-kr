# Lync Server 2013의 토폴로지 작성기에서 SQL Server 미러링 지원

 

_**마지막으로 수정된 항목:** 2012-09-26_

Lync Server 2013에서는 고가용성을 위한 백 엔드 서버 미러링이 지원됩니다. 토폴로지 작성기를 사용하여 미러링을 설정할 수 있습니다(미러링 모니터 서버 설정 포함).
