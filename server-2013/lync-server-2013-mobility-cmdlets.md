# 모바일 기능 cmdlet

 

_**마지막으로 수정된 항목:** 2012-06-20_

Lync Server 2010용 누적 업데이트: 2011년 11월에 추가된 모바일 기능을 관리할 수 있도록 모바일 기능용 cmdlet이 도입되었습니다. 이러한 cmdlet을 사용하여 Mobility Service 구성 및 사용자 정책과 같은 모바일 기능의 설정을 관리할 수 있습니다.
