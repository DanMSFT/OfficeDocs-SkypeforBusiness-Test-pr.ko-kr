# 고급 Enterprise Voice Cmdlet

 

_**마지막으로 수정된 항목:** 2012-03-21_

조직에서 Enterprise Voice를 구현할 때는 사용자 환경과 조직의 작업 효율성을 개선할 수 있는 추가 기능 및 응용 프로그램도 구현할 수 있습니다. 이러한 기능으로는 통화 허용 제어, E9-1-1(고급 9-1-1) 긴급 통화, 미디어 구성 설정 등이 있습니다.
