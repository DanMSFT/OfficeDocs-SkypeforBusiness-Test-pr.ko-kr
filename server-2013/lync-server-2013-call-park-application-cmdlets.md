# 통화 대기 응용 프로그램 Cmdlet

 

_**마지막으로 수정된 항목:** 2012-03-21_

통화 대기 응용 프로그램을 사용하면 통화를 대기시켰다가 다른 전화에서 재개할 수 있습니다. 아래에 나와 있는 cmdlet을 사용하여 통화 대기 파킹된 통화 번호 및 통화 대기 응용 프로그램에 대한 설정을 구성합니다.
