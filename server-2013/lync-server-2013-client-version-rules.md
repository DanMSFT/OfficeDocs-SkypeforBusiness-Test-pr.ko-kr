# 클라이언트 버전 규칙

 

_**마지막으로 수정된 항목:** 2012-12-11_

클라이언트 버전 정책은 클라이언트 버전 정책 규칙 집합으로 구성됩니다. 이러한 규칙은 사용자가 특정 클라이언트 및 클라이언트 버전을 사용하여 로그온을 시도할 때 수행할 동작을 정의합니다. 다음 절차에 따라 Lync Server 2013에 대한 클라이언트 버전 정책을 구성합니다.
