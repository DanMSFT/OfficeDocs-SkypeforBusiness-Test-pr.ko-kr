# 인증서 및 인증 Cmdlet

 

_**마지막으로 수정된 항목:** 2012-10-04_

인증서 및 인증 cmdlet는 서버 및 클라이언트 인증서 관리, 사용자 PIN(개인 식별 번호) 관리, 인터넷 정보 서비스에서 사용되는 SIP 도메인 및 Kerberos 계정 관리 등을 비롯한 다양한 범위의 작업을 수행합니다.
