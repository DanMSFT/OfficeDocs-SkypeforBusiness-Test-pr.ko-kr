# 클라이언트 버전 정책

 

_**마지막으로 수정된 항목:** 2012-12-11_

클라이언트 버전 정책은 전역적으로 또는 특정 사이트/풀/사용자 그룹에 대해 클라이언트 버전 규칙 집합을 적용하는 데 사용됩니다. 다음 절차에 따라 Lync Server 2013에 대해 클라이언트 버전 정책을 구성합니다.
