# 온라인 모임 추가 기능 사용자 지정

 

_**마지막으로 수정된 항목:** 2012-06-28_

Lync 2013용 온라인 모임 추가 기능은 Outlook 메시징 및 공동 작업 클라이언트 내의 모임 관리를 지원합니다. 여러 가지 방법으로 추가 기능 동작을 사용자 지정할 수 있습니다. 이러한 방법 중에는 로고와 텍스트를 온라인 모임 초대 본문에 추가하는 새로운 옵션도 포함됩니다.
