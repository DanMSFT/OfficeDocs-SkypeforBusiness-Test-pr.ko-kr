# 복제 Cmdlet

 

_**마지막으로 수정된 항목:** 2012-10-09_

복제 cmdlet은 Lync Server 복제를 모니터링하고 관리하는 방법을 제공합니다. 이러한 cmdlet을 사용하여 복제 설정을 구성하고 복제 진행 상태를 모니터링하며 서버에서 복제를 수동으로 수행할 수도 있습니다.

다음은 복제 관리와 직접 관련된 cmdlet 목록입니다.
