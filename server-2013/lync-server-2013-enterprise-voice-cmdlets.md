# Enterprise Voice Cmdlet

 

_**마지막으로 수정된 항목:** 2014-03-19_

Enterprise Voice는 Microsoft의 VoIP(Voice over IP)에 대한 구현입니다. Microsoft Lync Server 2013에서 Enterprise Voice를 관리하는 데 사용할 수 있는 cmdlet을 통해 다이얼 플랜(이전에는 위치 프로필이라고도 했음), 음성 정책, 경로 및 정규화 규칙을 만들고 수정할 수 있습니다.

## Enterprise Voice Cmdlet

Lync Server 제어판에서 Enterprise Voice에 적용되는 대부분의 관리 작업을 수행할 수 있습니다. 이러한 동일한 작업을 Lync Server 관리 셸에서 또는 스크립트(특정 작업을 자동화할 수 있음) 내에서 cmdlet을 사용하여 수행할 수 있습니다. 다음은 Enterprise Voice 관리에 직접적으로 관련된 cmdlet 목록입니다.
