# 고급 9-1-1 Cmdlet

 

_**마지막으로 수정된 항목:** 2012-06-20_

Microsoft Lync Server 2013에는 Enterprise Voice 솔루션의 E9-1-1(고급 9-1-1) 구현을 구현하고 관리할 수 있도록 해주는 cmdlet이 제공됩니다. 이러한 cmdlet을 사용하여 연결 지점을 실제 주소에 매핑하고 Enterprise Voice 사용자에게 필요한 설정을 구성하여 긴급 통화를 성공적으로 수행하고 위치를 응급 서비스 공급자에 자동으로 전송할 수 있습니다. Lync Server 제어판에서는 E9-1-1을 구성할 수 없습니다. cmdlet을 사용해야 합니다.
