# IM 및 현재 상태 Cmdlet

 

_**마지막으로 수정된 항목:** 2012-06-26_

Windows PowerShell을 통해 IM(인스턴트 메시징) 및 현재 상태 cmdlet을 사용하여 해당 클라이언트 기능을 관리할 수 있습니다. 전역, 사이트 또는 사용자별 범위에서 사용자에게 적용되는 현재 상태 정책을 설정할 수 있습니다. 또한 다양한 개인 정보 및 IM 기능을 구성할 수 있습니다.

## IM 및 현재 상태 Cmdlet

IM 및 현재 상태를 구성하려면 다음 cmdlet을 사용합니다.
