# 통화 허용 제어 Cmdlet

 

_**마지막으로 수정된 항목:** 2012-03-21_

CAC(통화 허용 제어)는 허용 가능한 품질의 실시간 오디오 또는 비디오 세션을 설정하기 위해 충분한 네트워크 대역폭이 있는지 여부를 확인합니다. CAC는 네트워크, 사이트, 서브넷 및 이들 간 상호 작용에 대한 제한과 설정을 구성하여 관리됩니다.
