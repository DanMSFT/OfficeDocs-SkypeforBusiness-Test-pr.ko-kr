# PSTN 연결 Cmdlet

 

_**마지막으로 수정된 항목:** 2012-10-09_

Enterprise Voice는 QoS(서비스 품질) 저하 없이 PSTN(공중 전화망)과의 통화를 허용하는 설정을 제공합니다. 이러한 설정은 Lync Server 관리 셸에서 제공되는 cmdlet을 통해 구성할 수 있습니다.
