# 음성 응용 프로그램 Cmdlet

 

_**마지막으로 수정된 항목:** 2012-10-09_

음성 응용 프로그램은 프런트 엔드 서버의 응용 프로그램 서비스에서 실행됩니다. 이러한 응용 프로그램을 사용하면 통화를 대기시켜 수신 전화와 발신 전화를 관리하고, 지정되지 않은 번호를 처리하고, Exchange 통합 메시징(UM)을 조정하고, 응답 그룹 응용 프로그램을 실행할 수 있습니다.
