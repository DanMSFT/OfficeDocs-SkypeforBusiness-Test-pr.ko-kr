# 전화 접속 회의 Cmdlet

 

_**마지막으로 수정된 항목:** 2012-03-21_

전화 접속 회의를 사용하면 "일반" 전화기(즉, 공중 전화망의 장치)를 통해 회의의 오디오 부분에 참가할 수 있습니다.

## 전화 접속 회의 Cmdlet

전화 접속 회의를 허용하지 않으려면 Set-CsConferencingPolicy cmdlet을 사용하여 EnableDialInConferencing 속성을 False로 설정함으로써 이 기능을 사용하지 않도록 설정할 수 있습니다. 기본적으로 모든 사용자는 전화 접속 회의를 포함하는 모임을 호스팅할 수 있습니다.
